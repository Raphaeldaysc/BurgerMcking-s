Como Usar os Cupons

  1. Escolha o cupom desejado
  2. Clique em "Usar Cupom"
  3. Copie o código do cupom
  4. Faça seu pedido no app ou site
  5. Cole o código na área de cupons antes de finalizar o pedido

Perguntas Frequentes

Posso usar mais de um cupom por pedido?

Não, apenas um cupom pode ser utilizado por pedido.

Os cupons são válidos para delivery?

Sim, a maioria dos cupons pode ser usada tanto para delivery quanto para retirada no restaurante.

Como faço para receber novos cupons?

Baixe nosso app e cadastre-se no Clube McKing's para receber cupons exclusivos regularmente!
